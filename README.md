# Aplicação de Quiz

Esta é uma aplicação de Quiz que permite aos utilizadores registarem-se, fazerem login e participarem num jogo de quiz. O programa acompanha a pontuação do utilizador e armazena as informações de login e pontuações num ficheiro TXT.

## Visão Geral do Projeto

1. A aplicação de Quiz tem as seguintes características:
   - Funcionalidade de registo e login de utilizador.
   - Página da conta do utilizador que exibe as seguintes opções:
     - Nome de utilizador.
     - Pontuação.
     - Iniciar jogo.
     - Sair.
   - Quando o jogador inicia o jogo, são apresentados os temas disponíveis para o quiz.
   - Após selecionar um tema, a aplicação exibe 10 perguntas de escolha múltipla de cada vez.
   - Cada tema possui mais de 10 perguntas de escolha múltipla, mas apenas 10 são apresentadas em cada sessão de jogo.
   - É fornecido um feedback imediato para cada resposta.
   - Cada resposta correta adiciona 5 pontos à pontuação geral do jogador.
   - Após completar o quiz, o programa retorna à página da conta.

## Características Adicionais do Projeto (opcional)

A implementação de qualquer uma das seguintes características adicionais concederá pontos extras para a nota final do projeto:

1. Limite de tempo para responder às perguntas (+5 pontos).
2. Funcionalidade de exclusão da conta (+5 pontos).
3. Opção de reiniciar a pontuação (+5 pontos).
4. Adição de 3 níveis de dificuldade (+10 pontos).
5. Recuperação de senha (+5 pontos).
6. Tabela de líderes (+5 pontos).
